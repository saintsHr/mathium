# gcd()

## Description:
returns the greatest common divisor of 2 given numbers

## Syntax:
```c
gcd(a, b);
```
### gcd()
- **Desc:** the actual function
- **Return:** *unsigned long long* ( the result of operation )

### a: 
- **Desc:** first operand
- **Type:** *long long*

### b: 
- **Desc:** second operand
- **Type:** *long long*
