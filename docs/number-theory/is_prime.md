# is_prime()

## Description:
returns true if the given number is prime, and false if not

## Syntax:
```c
is_prime(n);
```
### is_prime()
- **Desc:** the actual function
- **Return:** *bool* ( *true* if prime, *false* if not )

### n: 
- **Desc:** number to check if is prime
- **Type:** *unsigned long long*
